# Security Policy

## Supported Versions

| Component | Version | Supported |
|-----------|---------|-----------|
| Next.js   | 16.x    | Yes       |
| Node.js   | 24.x+   | Yes       |

## Reporting a Vulnerability

This is a private application. If you discover a security vulnerability, contact shaq@shaqhardy.com directly.

## Patching SLA

| Severity | Response Time |
|----------|---------------|
| Critical | Within 7 days |
| High     | Within 14 days |
| Medium   | Within 30 days |
| Low      | Next scheduled maintenance window |

Dependabot alerts and security updates are enabled. CodeQL code scanning runs on every push to main and weekly.
